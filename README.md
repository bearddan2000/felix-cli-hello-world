# felix-cli-hello-world

## Description
A POC for felix programming language.

## Tech stack
- g++
- make
- ocaml
- python

## Docker stack
- ubuntu:22.04
- rzuckerm/felix

## Build note
This will take 1+ hours to build

`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
